# ЛР №3 — Abstract Factory (Этап 2)

## Цель и задание
См. методичку: ЛР №3 «Фабричный метод и Абстрактная фабрика». fileciteturn0file0

## Реализация (Abstract Factory)
- Введён интерфейс **`IFigureFactory`** с методами:
  - `CreateCircle()`, `CreateSquare()`, `CreateTriangle()`
- Реализованы конкретные фабрики по цветам:
  - `RedFactory`, `BlueFactory`, `GreenFactory`
- **MainWindow** хранит ссылку на `IFigureFactory` и создаёт фигуры только через неё.

## Анализ
Плюс: одна иерархия фабрик вместо трёх. Новый цвет = новый класс фабрики, клиент почти не меняется.  
Минус: добавление новой фигуры требует изменения интерфейса `IFigureFactory` и всех фабрик.

## Контрольные вопросы (кратко)
1) Abstract Factory группирует несколько фабричных методов (по продуктам) в одном интерфейсе.
3) OCP: добавление нового цвета делается добавлением новой фабрики, без изменения продуктов/клиента (кроме выбора фабрики).
